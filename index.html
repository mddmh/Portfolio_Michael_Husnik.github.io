How to create a github page

  1. Go to
  2. Enter your email and click on "Sign up for GitHub"
  3. Create a new public repository - make sure to use your profile name as the repository name
  4. Open the repository in github desktop
  5. Create a new html file named index.html and the code should look like the next item.
  6.                     <html>
                        <body>
                            <h1> _Your_Name_ </h1>
                            <img src="_img_placeholder_" alt="_placeholder_">
                        </body>
                    </html>
  7. Next, import a image into the same repository as the one with your html file
  8. Replace the "_img_placeholder_" with the file name in the repository and the alt _placeholder_ with a description of the image
  9. After everything is set, commit your changes to the repository
  10. Don't forget to comment your changes!
  11. finally go to your github repository in the browser and publish your updated repository
